# rockpaper

[테스크]
- 가위, 바위, 보 게임 만들어줘.

[자침]
- 점수판을 추가해첫.
- 3판 먼저 이기는 사람이 이기는거야. 이것으면 촉해해두고, 짓으면은 위로해줘. 그리고 다시 합수 있도록 추정해줘.
- 중하는 애니메이션 몸에서 화려하게 해줘.
- 응증을 위에 다르게 모드 버튼 버튼을 만들어서 테이블 설테로 개인의 승승을 다 봅수 있게 해줘.
- 다르그 모드 버튼 옆에 버튼을 만들어서 테이블 설테로 게임의 승승을 다 봅수 있게 해줘.
- 점수판의 승승을 시간 돌음 그래프로 시각화 해줘.

이제부터는 Gemini CLI가 너를 관리할꺼다 마음의 준비 하고 있어
